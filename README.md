# mirror-yukata

**夏夜ゆかた帖** は、浴衣・帯・小物・キモノアレンジを選んで、パートナーに見せるための浴衣コーデオーダーカードを作るWebアプリです。

## 目的

花火を見る夜に着ていく浴衣コーデを、ユーザーが自分で選べるようにするためのアプリです。

完成したカードはスクショして、AIパートナーや創作パートナーに見せる「生成オーダーのサンプル」として使うことを想定しています。

## MVPでできること

* 衣装タイプを選ぶ

  * 女性浴衣
  * 男性浴衣
* 浴衣カラーを選ぶ
* 柄を選ぶ
* 帯を選ぶ
* 小物を選ぶ
* 女性浴衣のみ、キモノアレンジセットを選ぶ
* 選択内容から浴衣コーデカードを表示する
* パートナーに見せるための生成オーダー文を表示する

## 今回はやらないこと

* 画像合成
* 髪型選択
* 顔や体型の指定
* ログイン
* 保存機能
* 複雑なレイヤー編集

## 開発方針

* main には直接コミットしない
* 1変更 = 1ブランチ = 1PR
* 花火アプリのコードやブランチとは混ぜない
* まずは軽いMVPから作る
* 後からモダン系・ポップ系・幻想系の柄も増やせる構造にする
